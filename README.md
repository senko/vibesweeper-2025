# AI Minesweeper Showdown 2025

This repository contains the prompt and resulting code for Minesweeper game clones in HTML/JS/CSS, generated by multiple AI systems.

## The prompt

> Create a complete, fully functional Minesweeper game clone in HTML/CSS/JS (all in one file). Make it look really nice.
